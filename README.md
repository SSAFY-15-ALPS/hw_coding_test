# 코딩테스트 연습

## 프로젝트 구조
```
CodingTest/
├── src/
│   └── (문제별 Java 파일)
├── build.gradle
├── settings.gradle
└── gradlew
```

## 사용 방법

### IntelliJ에서 실행
1. `File` → `Open` → CodingTest 폴더 선택
2. Gradle 프로젝트로 자동 인식됨
3. Java 파일 열고 우클릭 → `Run` 또는 `Ctrl+Shift+F10`

### 터미널에서 실행
```bash
# 빌드
./gradlew build

# 특정 클래스 실행
./gradlew run --args="ClassName"
```
