# 알고리즘 문제 풀이 저장소

SSAFY 알고리즘 스터디를 위한 코딩테스트 문제 풀이 저장소입니다.

## 프로젝트 정보

- **언어**: Java 8
- **빌드 도구**: Gradle
- **플랫폼**: 백준(Baekjoon), SWEA(SW Expert Academy)

## 폴더 구조

```
├── 백준/
│   ├── Bronze/
│   ├── Silver/
│   └── Gold/
│
└── SWEA/
    ├── D1 ~ D5/
    └── Algorithm_track_level_*/
```

- **백준**: 난이도별(Bronze, Silver, Gold, Platinum) 폴더로 분류
- **SWEA**: 난이도(D1~D5) 및 알고리즘 트랙별로 분류

## 사용 방법

### IntelliJ IDEA

1. `File` → `Open` → 프로젝트 폴더 선택
2. Java 파일 열고 `Ctrl+Shift+F10` 또는 우클릭 → `Run`

### 터미널

```bash
./gradlew build
```
